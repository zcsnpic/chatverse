# ChatVerse - 虚拟聊天空间

一个支持多对象聊天的虚拟聊天网站，可以对每个聊天对象进行关键记忆、身份和角色的设定。

## ✨ 特性

- 🤖 **双轨制消息系统**：AI角色输出包含语音（台词）和动作/场景两部分
- 💾 **本地存储**：所有数据存储在本地，保护隐私
- 🎭 **多角色支持**：创建多个不同性格的AI角色
- 🧠 **记忆系统**：为每个角色设置关键记忆，影响对话内容
- 🔄 **上下文重置**：一键重置对话上下文，保留角色设定
- 🎨 **Midnight Galaxy 主题**：沉浸式的宇宙风格UI设计
- 🔌 **多AI模型支持**：
  - DeepSeek
  - OpenAI (GPT-4o, GPT-3.5)
  - Anthropic (Claude 3.5)
  - Google Gemini
  - Ollama (本地部署)

## 🚀 快速开始

1. 克隆仓库
2. 使用任意静态服务器打开 `index.html`
3. 在设置页面配置 AI API Key
4. 开始创建你的第一个虚拟角色！

## 📝 使用说明

### 创建角色
1. 点击底部的"创建"按钮
2. 填写角色名称、身份和角色设定
3. 添加关键记忆（可选）
4. 点击"创建角色"

### 开始聊天
1. 点击首页的角色卡片
2. 在输入框中输入消息
3. AI角色会以双轨制形式回复：
   - **语音**：角色的台词
   - **动作/场景**：角色的动作和环境描述

### 重置上下文
1. 在聊天页面点击右上角菜单
2. 点击"重置上下文"
3. 确认操作

这会清除当前对话的上下文影响，但保留角色的设定和关键记忆。

## ⚙️ AI模型配置

1. 进入"设置"页面
2. 点击"AI 模型"
3. 选择提供商和模型
4. 输入 API Key
5. 保存设置
